# REID Workspace

这个仓库用于整理 ybj 工作区里适合放进 Git 的轻量内容：源码、配置、脚本和说明文档。

默认纳入版本管理的部分：

- TVI-LFM 的源码、配置、训练脚本和项目代码
- 手动权重入库脚本与中文说明
- 训练前预检脚本与辅助文档

默认不纳入版本管理的部分：

- 模型权重
- 数据集和派生数组文件
- 训练日志、检查点和输出目录
- 本地 Codex 运行时、临时目录和缓存文件

详细的上传清单、排除理由和推送步骤见 UPLOAD_MANIFEST_ZH.md。

当前工作区还有两个结构性注意点：

1. 根目录的 .git 目前是空目录，不是一个有效的 Git 仓库。
2. TVI-LFM 自带独立的 .git；如果你要把整个 ybj 作为一个仓库上传，需要先把这个嵌套仓库元数据备份或移开。
