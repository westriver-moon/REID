生成日期：2026-06-13

说明：

- 表中数值均为百分比。
- `Final` 指训练脚本最终保存的评估结果，通常是对所选 checkpoint 的 10-trial 或配置指定 trial 评估。
- `Best epoch` 来自训练过程中的 `history.csv`，多数是低 trial 监控结果，只用于选点和趋势判断，不应直接当作最终论文指标。
- `RGB-only`、`IR-only`、`same-modality`、`cross-modality` 协议不同，不能直接混在一起比较绝对值。

## 一、关键结论

1. 当前跨模态 SYSU fine-tune 中，`vcm_llcm_partial_shared_finetune_ep20` 是目前这一批外部预训练路线的最好结果：All mAP 26.84 / Rank-1 23.31，Indoor mAP 39.68 / Rank-1 26.79。
2. `VCM + LLCM` 外部预训练后 fine-tune 明显优于 `LLCM only`：All mAP 提升 2.08，Rank-1 提升 1.48；Indoor mAP 提升 2.26，Rank-1 提升 2.29。
3. 双编码器 adapter 系列中，`B4_h192_sampler_4rgb4ir_ep30` 的 Indoor Rank-1 最高，为 27.52；但 All mAP 25.70 仍低于外部预训练 fine-tune 的 26.84。
4. Shared Transformer Head 没有超过 adapter / external fine-tune 路线：2-layer head 尚可，4-layer head 明显退化。
5. Patch 消融中，单次实验里 multi-branch 的训练中 best epoch 更高；但三次 seed 的 final 均值上，`C_singlebranch_288x144` 略高于 `B_multibranch_288x144`，说明多分支 patch 的收益还需要更严格地复核。

## 七、当前比较建议

1. 跨模态主线优先参考 `sysu_external_pretrain/vcm_llcm_partial_shared_finetune_ep20`，因为它在 All mAP / Rank-1 上领先当前保留实验。
2. 若关注 Indoor Rank-1，`sysu_rgb_ir_dual/B4_h192_sampler_4rgb4ir_ep30` 仍值得作为对照，因为其 Indoor Rank-1 达到 27.52。
3. 外部预训练直接迁移到 SYSU 的指标很低，说明域差异仍然明显；它更适合作为初始化来源，而不是直接作为最终模型。
4. Patch 消融还不能强结论地证明 multi-branch 一定优于 single-branch；目前更稳妥的表述是：multi-branch 在部分 seed 的 best epoch 有优势，但 final 多 seed 均值不占优。
5. Shared Transformer Head 目前没有证明比 gated adapter 更好，后续若继续做，应先解决 hard positive / hard negative 排序问题，而不是单纯加深 head。
